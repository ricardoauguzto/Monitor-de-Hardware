# Monitor-de-Hardware
Projeto simples de monitoramento de hardware desenvolvido em Java.
O sistema exibe em tempo real o uso da CPU e memória RAM através de uma interface minimalista e leve, com foco em aprendizado de monitoramento de sistema e interface gráfica desktop.
